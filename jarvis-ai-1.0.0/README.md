# JARVIS AI - Next-Gen Voice Assistant

Inspired by Iron Man’s JARVIS and the Manina Labs demo.

## Features
- **Voice Interaction**: Wake word "Jarvis", natural conversation with LLM.
- **Futuristic UI**: Dynamic particle orb, system HUD with real-time diagnostics.
- **System Monitoring**: CPU, RAM, diagnostics.
- **Commands**: Diagnostics, web search, tasks, jokes, etc.
- **Offline-capable** with local models.

## Tech Stack
- Python 3.11+
- SpeechRecognition + pyttsx3 or Piper TTS
- Ollama or Groq for LLM
- Pygame for particle effects
- psutil for system stats

## Installation
1. `git clone https://github.com/ryanpett26-pixel/jarvis-ai.git`
2. `cd jarvis-ai`
3. `bash setup.sh`
4. `python main.py`
